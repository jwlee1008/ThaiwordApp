# Privacy Policy Draft

Last updated: 2026-05-06

`KorThai Words` is a local-first Korean vocabulary practice app for Thai learners.

## Data Collection

The app does not require login and does not send study records to a server in the current MVP version.

The following learning data is stored locally on the user's device:

- Favorite words
- Wrong-answer note
- Seen words
- Correctly answered words

This data is used only to show learning progress and improve the in-app practice experience.

## Network Use

The current MVP app uses bundled vocabulary data and does not need a network connection for normal study features.

## Third-Party Data Source

Vocabulary data is based on the National Institute of Korean Language Korean Basic Dictionary Open API. Source and license information is available inside the app under Settings > Source and License.

## Data Deletion

Users can delete local study records from the Settings tab by using the reset study state action.

## Future Changes

If login, cloud backup, paid content, analytics, or push notifications are added later, this policy must be updated before release.
