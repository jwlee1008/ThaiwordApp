# App Context Review

2026-05-06 중간 점검 메모입니다.

## Current Direction

앱의 첫 화면은 카테고리 탐색보다 난이도별 학습 덱이 더 적합하다.

이유:

- 태국어 학습자 입장에서는 "어디부터 시작할지"가 먼저 필요하다.
- 카테고리 16개를 첫 화면에 모두 보여주면 MVP가 정보 탐색 앱처럼 보인다.
- 현재 데이터는 `levelKo`가 있으므로 초급/중급/고급 흐름으로 확장하기 쉽다.
- 카테고리는 버리지 않고 내부 태그/검색/향후 필터로 유지하는 편이 좋다.

## Changes Applied

- 학습 탭 첫 화면을 `난이도` 중심으로 변경
- 현재 데이터는 `초급` 덱에 모음
- `중급`, `고급`은 준비 중 상태로 표시
- 즐겨찾기와 오답노트는 빠른 복습 영역으로 유지
- 카드 연습에서 단어를 열거나 넘기면 학습한 단어로 기록
- 오답 기록은 이미 본 단어 기록과 함께 안정적으로 저장

## Product Notes

현재 MVP 맥락은 다음처럼 정리된다.

- 앱 목적: 태국어 사용자가 한국어 기초 단어를 반복 연습한다.
- 첫 사용 흐름: 초급 덱 선택 -> 카드 연습 또는 퀴즈 -> 진행률/오답/즐겨찾기 누적
- 검색 탭 역할: 이미 본 단어를 빠르게 다시 찾거나 뜻을 확인한다.
- 설정 탭 역할: 학습 상태, 기록 초기화, 출처/라이선스 확인

## Later Improvements

- 중급/고급 데이터팩을 추가하면 같은 화면 구조를 유지한다.
- 카테고리는 첫 화면이 아니라 덱 안의 필터 또는 검색 필터로 부활시키는 편이 좋다.
- 퀴즈 유형을 늘릴 때도 첫 화면 버튼을 늘리지 말고 단어 목록 내부의 학습 모드로 넣는다.
